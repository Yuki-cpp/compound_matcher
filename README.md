

# Install

## Method 1

```shell
git clone https://github.com/Yuki-cpp/compound_matcher.git
cd compound_matcher
pip install .
```

## Method 2


```shell
pip install https://github.com/Yuki-cpp/compound_matcher/archive/master.zip
```

# Run

## Linux

```shell
compound_matcher -h
```

## Windows
```shell
python -m compound_matcher.find_matches -h
```
